# DroidPerf
DroidPerf — a free, open-source Android game performance profiler (alt to PerfDog). Tracks live FPS, 1%/10% lows, CPU/GPU clocks, RAM, battery, and thermals via ADB. Click Start, play, click Stop — get an AI-generated bottleneck report. Built end-to-end with Codex (GPT-5.6); runtime analysis uses MiniMax M3 via NVIDIA NIM.
